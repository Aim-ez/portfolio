# Personal Portfolio

My personal portfolio website, built from scratch using HTML, CSS, and JavaScript.

The goal of this project is twofold:
- Showcase my software development projects and experience.
- Refresh and strengthen my front-end development skills by building without frameworks.

- CSS3
- JavaScript
- GitHub Pages
